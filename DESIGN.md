# Twilight Design System

Cymek's visual identity is the **Twilight** variant of Mistral AI — purple-blue instead of orange-red.

## Brand Voice
- Clean, technical, confident
- "Your docs, your data, your edge"

## Color Palette
- **Primary**: #6B4FF5 (purple-blue)
- **Primary Dark**: #5538D6
- **Primary Light**: #8B7BF7
- **Surface**: #F8F6F0 (cream)
- **Surface Alt**: #EDE9E0
- **Text Primary**: #1A1525
- **Text Secondary**: #6B6580
- **Border**: #D4CFC8
- **Success**: #22C55E
- **Warning**: #F59E0B
- **Error**: #EF4444

## Typography
- **Headings**: PP Editorial Old (serif) — display, h1–h3
- **Body**: Inter (sans-serif) — h4–h6, body, code, labels
- **Monospace**: JetBrains Mono — code blocks, inline code

## Spacing (4px grid)
- xs: 4px, sm: 8px, md: 16px, lg: 24px, xl: 32px, 2xl: 48px, 3xl: 64px

## Radius
- sm: 4px, md: 8px, lg: 12px, xl: 16px

## Shadows
- sm: `0 1px 2px rgba(26, 21, 37, 0.06)`
- md: `0 4px 12px rgba(26, 21, 37, 0.08)`
- lg: `0 8px 24px rgba(26, 21, 37, 0.12)`

## Component States
- Focus ring: `0 0 0 2px #6B4FF5`
- Disabled: opacity 0.4
- Error: border-color #EF4444
